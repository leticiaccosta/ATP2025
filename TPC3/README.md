## Resumo
Este trabalho prático teve como objetivo desenvolver uma aplicação em Python capaz de manipular listas de números inteiros através de um menu interativo. A aplicação permite ao utilizador criar e ler listas, calcular a soma e média, encontrar o maior e menor elemento, verificar se a lista  está ordenada e procurar elementos específicos. 
Durante o desenvolvimento do trabalho, foram trabalhados conceitos fundamentais da programação, nomeadamente: estruturas de repetição, estruturas condicionais, definição e utilização de funções, listas e manipulação de dados. Como tal, surgiram algumas dificuldades, sobretudo relacionadas com: a compreensão do funcionamente dos indices das listas, o que levou incialmente a resultados incorretos ao procurar elementos; a necessidade de interromper a pesquisa ao encontrar um elemento, já que inicialmente a função percorria toda a lista mesmo após encontrar o valor; a gestão da execução contínua do programa, uma vez que incialmente a aplicação terminava após executar uma funcionalidade. Tudo isto foi superado através da identificação do erro e procurar perceber a origem do problema e, posteriormente, aplicar a solução mais adequada, como por exemplo, introduzindo ciclos while.
Apesar disso, estas dificuldades foram essenciais para consolidar a compreensão de conceitos fundamentais da programação.
